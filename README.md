# vehicleAdvertisementAssignment

This is the Implementation of a REST service for managing listings for online advertising service.The required functionalities were originally as follow:
•	Create a listing. All the created listings should have state draft by default;
•	Update a listing;
•	Get all listings of a dealer with a given state;
•	Publish a listing;
•	Unpublish a listing.
Some crud functionalities about dealer have also been added to ease some tests and verification: 
•	Create a Dealer 
•	Get  all  Dealer
•	Get  a  Dealer details
•	Update a Dealer
•	Delete a Dealer
